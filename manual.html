NURBS.js Manual

Welcome to NURBS.js 2D!

Here you will find the documentation for what commands are available, and how to use the canvas-mouse interface.

Please note that NURBS.js is a work in progress and will be changing frequently. Check out the GitHub page to learn more about the project and to find the source code.


Canvas-Mouse Interface

There are event handlers that are designed for click and drag on our canvas, here is what is currently implemented.

Left Click

If you left click on or near some piece of geometry on the canvas it will highlight yellow and be selected.

If you hold SHIFT while you left click, it will allow you to select multiple pieces of geometry.

If you click away from any geometry while it is selected and you do not have SHIFT held, then it will deselect all geometry that has already been selected.

Right Click and Drag

To move around the canvas you simply can right click and drag on the canvas.

Command Line

The system is currenty based on a command line interface. The command line allows you to start and give inputs to commands. Some commands may require you to have a geometry selected before you start them.

Your interactions with the command line will be saved to a history and displayed in the block above the command line.

Many of these commands are based on inputting points into the command line until a point and click mouse interface is added to the system. The command line will take a particular format for a point that should be types as

        x,y,z

If it is not formatted this way, it will fail to parse you point and prompt you to enter the coordinates again.

Move

This command requires you to have some geometry selected before starting the command.

The move command will prompt you to enter two points. These two points will create a vector for which to move your selected geometry by. The prompts will ask you to enter a start point for your move and then an end point for your move.

Line

This command will ask for a start and end point for a line and then create that line and add it to the canvas.

Circle

The circle command will create a circle and apply it to the canvas. Currently this function will ask you to input a center point for the circle and then it will ask for a radius to the circle.

Rectangle2

The rectangle2 command creates a rectangle based on two corner points. The command will prompt you to input two points the correspond to the two opposite corners of the rectangle.

SelNone

This command is an alternate way to deselect all geometry.

Polygon

The polygon command allows you to create a regular n-sided polygon. The command will prompt you to first input the center you want your polygon to be generated around, then prompt you to input the number of sides your polygon should have, and then it will ask you for a corner radius for you polygon.

In this case, corner radius is devined as the distance between the center of your polygon and the vertices of each side.

Bounding Box

The bounding box command will find the extremes of the geometry that you have selected and create a rectangle around them.

You must have some geometry selected before running this command.
